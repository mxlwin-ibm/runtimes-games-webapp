# Foosball Tournament MCP System

## Goal

Build a complete Foosball Tournament Management System using:

- FastAPI
- MongoDB
- FastMCP
- React
- MCP Clients (Bob IDE, Claude Desktop, Cursor)

---

## Architecture

User
↓
MCP Client (Bob IDE)
↓
FastMCP Server
↓
FastAPI Backend
↓
MongoDB

---

## Features

### Team Management

- Add Team
- View Teams
- Delete Team

### Match Management

- Schedule Match
- Update Result
- View Matches

### Points Table

- Calculate standings
- Rank teams

### MCP

- Tools
- Resources
- Prompts

### UI

- Teams
- Matches
- Point Table
